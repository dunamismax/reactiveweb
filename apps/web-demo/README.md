# web-demo

Flagship ReactiveWeb app that demonstrates the full stack contract in one place:

- React Router framework mode (SPA-first)
- React 19 + TypeScript UI
- Tailwind + shadcn-style primitives
- Zod input/env validation
- Auth.js configuration patterns
- Drizzle/Postgres schema integration via shared packages

## Commands

```bash
bun run dev
bun run build
bun run typecheck
```
